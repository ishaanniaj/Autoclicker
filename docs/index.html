Install in under a minute

No Terminal, no setup — just download and drag.

Download Autoclicker.dmg and open it.

Drag Autoclicker onto the Applications folder.

Open it from Applications — first time, right-click → Open → Open. (If macOS says it can't verify the developer, go to System Settings → Privacy & Security → Open Anyway.)

Enable it under System Settings → Privacy & Security → Accessibility so it can click, then quit and reopen.

Frequently asked questions

 

Is this autoclicker free?

Yes — it's completely free and open source.

Does it work on Windows?

No. This is a macOS app for Apple Silicon Macs (M1, M2, M3, M4). It does not run on Windows, Chromebooks, or older Intel Macs.

Is it safe?

Yes. It's open source — you can read all the code on GitHub. The one-time "unverified developer" prompt appears because the app isn't paid-registered with Apple, not because anything is wrong with it.

How do I change the click speed?

Type a number in the Click interval boxes (hours / minutes / seconds / milliseconds). For example, 100 ms means about 10 clicks per second.

How do I start and stop it?

Press the Start button, or tap the ` (backtick) key — it works even when another app is in front.
