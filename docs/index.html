A free, lightweight auto clicker for macOS.

⬇︎ Download for Mac

Free • Apple Silicon Macs (M1, M2, M3, M4) • macOS 11 or later

What it does

Autoclicker for Mac clicks your mouse automatically at a speed you choose — great for games, repetitive tasks, and testing. It's a small, no-frills auto clicker: open it, set the interval, and go.

  • Adjustable click speed — from many clicks per second to one every few minutes
  • Left, middle, or right mouse button
  • Single or double click
  • Run continuously or stop after a set number of clicks
  • Global hotkey — tap the ` (backtick) key to start or stop from any app

How to install the autoclicker on Mac

  1. Download Autoclicker.dmg and open it.
  2. Drag Autoclicker onto the Applications folder.
  3. Open it from Applications — the first time, right-click the app and choose Open, then Open again. (If macOS says it can't verify the developer, go to System Settings → Privacy & Security and click Open Anyway.)
  4. Enable it under System Settings → Privacy & Security → Accessibility so it can click, then quit and reopen.

Frequently asked questions

Is this autoclicker free?

Yes — it's completely free and open source.

Does it work on Windows?

No. This is a macOS app for Apple Silicon Macs (M1, M2, M3, M4). It does not run on Windows, Chromebooks, or older Intel Macs.

Is it safe?

Yes. It's open source — you can read all the code on GitHub. The one-time "unverified developer" prompt appears because the app isn't paid-registered with Apple, not because anything is wrong with it.

How do I change the click speed?

Type a number in the Click interval boxes (hours / minutes / seconds / milliseconds). For example, 100 ms means about 10 clicks per second.

How do I start and stop it?

Press the Start button, or tap the ` (backtick) key — it works even when another app is in front.
